Freader

Now that Google Reader is over, why not use an open source Feed Reader ! Freader provide you a simple page with all your feeds and the last item in them. You cannot read articles inside the application for now but it's on the TODO list !

You can see it in action in just a little Heroku app here ! Account creation take about 10 seconds, just try it !

Features

  • All your feeds in a clear (and flat) UI
  • All written with Node.js and Angular.js
  • User managment (creation, removing)
  • Feed caching (one hour by default)

Requirement

  • Node.js
  • MongoDB

Installation

  • Clone this repo
  • Start your MongoDB server (modify the adress if you need in the config file)
  • Start the server with 'npm start'
  • You can now create a user and start adding feeds by connecting to localhost:3000 (you can change the port inside the config file)

Used

  • Angular.js
  • Bootstrap
  • FlatUI by Designmodo
